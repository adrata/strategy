# Speedrun

Speedrun is Adrata's quick actions and communication platform designed for rapid execution of common tasks. It provides instant access to frequently used actions, streamlined communication tools, and quick workflow execution.

## Overview

Speedrun enables rapid task execution and communication:

- **Quick Actions** - Instant access to frequently used actions
- **Communication Tools** - Streamlined messaging and calling capabilities
- **Rapid Workflows** - Execute quick workflows and processes
- **Context Switching** - Seamlessly switch between different tasks
- **Mobile Optimized** - Optimized for mobile and desktop use

## Getting Started

### 1. Access Speedrun

1. Log in to your Adrata workspace
2. Navigate to **Speedrun** from the main menu
3. You'll see the Speedrun dashboard with quick actions

### 2. Explore Quick Actions

1. Browse available quick actions in the dashboard
2. Click on actions to execute them
3. Use the search bar to find specific actions
4. Customize your action shortcuts

### 3. Set Up Communication

1. Configure your communication preferences
2. Set up voice calling integration
3. Configure email templates
4. Test your communication tools

### 4. Create Quick Workflows

1. Go to **Workflows** in Speedrun
2. Choose from template workflows or create custom ones
3. Set up workflow shortcuts
4. Test and activate your workflows
