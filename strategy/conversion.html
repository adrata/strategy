The 6 Game UX Lenses

From Scott Jenson + Raph Koster. How games create engagement without users knowing they're being taught.

1. Story vs Narrative

Story is author-generated. Narrative is user-generated. Create connected arcs so prospects construct: "This company understands my situation."

2. Games Are Fractal

Every action contains nested layers. Layer feedback at click, answer, section, and completion levels.

3. The Learning Loop

Fun = Learning. Master ONE mechanic. Same gesture everywhere = cumulative learning.

4. Affordances

Prompt things they ALREADY know. Grace notes build muscle memory toward the meeting booking action.

5. Hintiness

Move users to NEW loops. Don't let them bottom-feed on free content. Subtle pulsing on CTAs.

6. Pacing

Start with ONE question, ONE button. Everything reveals progressively.

Core Psychological Principles

Curiosity Gap
Loewenstein, 1994

"What % of VPs hit quota?" User guesses → reveal "43%" → They NEED to know if they were right.

Zeigarnik Effect
1927

Incomplete tasks remembered 90% better. Show score, blur "Critical Gaps." Brain won't let them stop.

Goal Gradient
Hull, 1932

Progress bar that accelerates: 60%... 85%... 97%... The acceleration is felt.

Endowed Progress
Nunes & Drèze, 2006

"3 of 8 sections pre-filled." They feel 37% done before starting. +79% completion.
